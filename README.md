👻 Ghost Maze Nightmare

Ghost Maze Nightmare is a simple horror-themed maze game built using HTML, CSS, and JavaScript.

🎮 Play the Game

🔗 Live Demo: https://ashvithareddy22.github.io/ghost-maze/

🕹️ How to Play

* Use **Arrow Keys** or **WASD** to move.
* Explore the haunted maze.
* Collect all **3 keys** hidden in different locations.
* Reach the **exit door** after collecting all keys.
* Avoid the ghost that continuously chases you.
* Touching a wall or getting caught by the ghost ends the game.

✨ Features

* Horror-themed maze environment
* Ghost enemy with multiple difficulty levels
* Key collection system
* Exit door unlocking mechanic
* Easy, Medium, and Nightmare modes
* Built with pure HTML, CSS, and JavaScript
* No external libraries required

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
